# Agent Prompt & Skill

本仓库存储各类场景的 Agent Skills 和 System Prompts，按使用场景划分文件夹。

## 目录结构

```
├── software-development/     # 软件开发场景
│   └── sdd/                 # SDD 软件设计文档
├── code-review/             # 代码审查场景
├── documentation/            # 文档编写场景
└── general/                 # 通用场景
```

## 使用说明

- **System Prompts**：定义 Agent 的身份、职责、工作流和约束
- **Skills**：按需加载的可复用任务能力，定义触发条件和执行步骤
